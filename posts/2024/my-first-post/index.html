在Android 11 上使用 LLDV 调试原生程序

September 28, 2024 One-minute read
XgDebug
LLDB • ANDROID

手机端 Link to heading

PUSH 调试服务器到手机 Link to heading

adb push lldb-server /data/local/tmp
chmod 755 /data/local/tmp/lldb-server

启动调试器服务 Link to heading

/data/local/tmp/lldb-server platform --listen "*:8888" --server

电脑端 Link to heading

端口转发 Link to heading

adb forward tcp:8888 tcp:8888

启动 LLDB Link to heading

.\lldb.exe

查看支持平台 Link to heading

platform list

选 ANDROID 平台 Link to heading

platform select remote-android

连接到手机 手机序列号: 9643e0ec0604 要换成当前调试的手机,使用 adb devices 查看序列号 Link to heading

platform connect connect://9643e0ec0604:8888

查看当前正在运行的进程 Link to heading

platform process list

附加上去 Link to heading

attach 9053

下断点 Link to heading

b send

c

thread list
