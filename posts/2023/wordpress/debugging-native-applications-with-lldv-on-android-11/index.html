Debugging native applications with LLDV on Android 11

August 11, 2023 One-minute read
ANDROID

Mobile Link to heading

PUSH debug server to mobile Link to heading

adb push lldb-server /data/local/tmp
chmod 755 /data/local/tmp/lldb-server

Start the debugger service Link to heading

/data/local/tmp/lldb-server platform --listen "*:8888" --server

Computer side Link to heading

Port forwarding Link to heading

adb forward tcp:8888 tcp:8888

Start LLDB Link to heading

. \lldb.exe

View supported platforms Link to heading

platform list

Select ANDROID platform Link to heading

platform select remote-android

Connect to the phone Phone serial number: 9643e0ec0604 To change to the current debugging phone, use adb devices to check the serial number Link to heading

platform connect connect://9643e0ec0604:8888

View the currently running process Link to heading

platform process list

Attach to Link to heading

attach 9053

Breakpoint Link to heading

b send

Run up Link to heading

c

View the list of threads Link to heading

thread list

bt
